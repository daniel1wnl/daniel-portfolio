## Overview

A web application that fetches and displays leaderboards for Riot Games' titles, deployed using Google Cloud Run for scalability.

## Problem

Tracking competitive rankings across different regions often requires multiple API calls and manual collation. This project automates that process.

## What I Built

- Integration with Riot Games API.
- Backend services using Node.js.
- Automated deployment pipeline to Google Cloud Run.

## What I Learned

- Managing API rate limits.
- Containerization with Docker for Cloud Run.
- Handling environment variables securely in a cloud environment.
